This project is a complete Spotify Top 50 World Analytics Dashboard built in Power BI, visualizing global song rankings, artist performance, popularity scores, audio features, and explicit content distribution.

It covers insights from Spotify’s Top 50 global chart featuring:   
       
- Taylor Swift       
   
- The Weeknd    
      
- Drake  
 
- Ed Sheeran 
  
- Dua Lipa

  The dashboard provides a clean, modern interface with insights into popularity, song duration, explicit content, artist trends, and audio features.

## **📌 Project Overview**

This Power BI dashboard analyzes Spotify’s global Top 50 songs using custom DAX calculations, data modeling, and modern UI design inspired by real-world music analytics platforms.
 
*The visualizations highlight:*
 
- ⭐ Top Songs & Popular Artists

- 📈 Popularity Metrics (Avg, Max, Min)

- 🅴 Explicit vs Non-Explicit Song Analysis

- ⏱️ Song Duration Comparison (Avg, Max, Min)

- 🎧 Audio Features — Energy, Danceability, Valence, BPM

- 🏆 Position #1 Songs & Artist Performance

- 🎶 Artist Metrics — Song Count, Popularity, Trends

- 🔍 Feature-based trend comparisons

## **📊 Features**

- ✔ Detailed Artist Insights
  
  Song counts, popularity scores, position #1 hits.

- ✔ Modern UI
  
  Spotify-themed color palette with glowing cards and dark interface.

- ✔ Audio Features Breakdown
  
   Energy, Danceability, Valence, BPM visualized in charts.

- ✔ Explicit vs Non-Explicit Insights

  Song classification with DAX-based percentage logic.

- ✔ Popularity Metrics

   Average, Maximum, Minimum popularity visuals.

- ✔ Song Duration Analysis

  DAX measures converting duration (ms → minutes).

- ✔ Multi-page Dashboard
  
  Overview, Song Analytics, Artist Insights, Audio Features.
